# Composable Sheep!

Repository used for the "Composable Sheep: The Creative Coding Epilogue" presentation.

## Sheep Sketches
Find inside the *sheepSketches* module several playground screens to play around with Spirals, Paths, Grids, Noise, Shaders...AND **SHEEP**! üêë
